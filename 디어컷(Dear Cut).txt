## 🌟 헤어스타일 컨시어지 플랫폼 구체화: "고민 없이 예약, 설명 없이 완성"

### 1단계: AI 스타일링 분석 및 시각화 (개인화 엔진)

이 단계는 사용자의 현재 상태를 분석하고, 미래의 스타일을 예측하여 시각적인 만족도를 제공하는 핵심입니다.

| 핵심 기능 | 기술적 구현 목표 | 사용자 경험 (UX) |
| :--- | :--- | :--- |
| **현재 상태 분석** | **Vision AI + 딥러닝:** 업로드된 사진에서 얼굴형, 이목구비 비율, 현재 모발 길이/숱/손상도를 초정밀 분석합니다. | 사진 한 장 업로드로 끝. "당신의 얼굴은 OOO형이고, 모발 손상도는 중입니다."라는 보고서 즉시 제공. |
| **맞춤형 추천** | **GAN 기반 스타일 합성:** 분석된 얼굴형과 트렌드 데이터를 기반으로, 가장 어울리는 헤어스타일 **5가지**를 추천합니다. (사용자가 직접 스타일을 선택할 수도 있음) | 추천 스타일이 **사용자의 얼굴에 합성**된 이미지(Virtual Try-On)를 제공. 스타일 선택에 대한 고민을 제거. |
| **디자이너 팁** | 선택된 스타일에 대해 "이 머리는 곱슬이 심하면 유지하기 어렵습니다"와 같은 **현실적인 제약 조건**을 미리 안내합니다. |

### 2단계: 미용실 필터링 및 추천 (간편 예약 엔진)

AI 추천 스타일을 **실제로 잘 시술할 수 있는** 미용실을 연결하여 만족도를 높이는 단계입니다.

| 핵심 기능 | 기술적 구현 목표 | 사용자 경험 (UX) |
| :--- | :--- | :--- |
| **스타일별 매칭** | **시술 능력 데이터베이스(DB):** 미용실/디자이너별로 **레이어드 컷, 옴브레 염색, 아이롱 펌** 등 시술별 전문도와 포트폴리오 데이터를 구축합니다. | 1단계에서 선택된 스타일에 대해 전문성이 높은 미용실을 **자동으로 상위 랭크**합니다. |
| **필터 및 정렬** | **지도 API 및 평점 연동:** '현재 위치 기준 거리순', '사용자 리뷰 평점순', 'AI 추천 스타일 시술 건수순' 등으로 필터링 및 정렬 옵션을 제공합니다. | 사용자는 복잡한 검색 없이 **"집에서 가장 가깝고, 이 머리를 잘하는 곳"**만 볼 수 있습니다. |
| **자동 예약 연동** | 선택된 미용실의 **실시간 빈 시간표**를 캘린더 형태로 보여주어, 클릭 한 번으로 예약할 수 있게 합니다. |

### 3단계: AI 요청서 자동 생성 (LLM 기반 요청서 작성)

사용자가 원하는 바를 명확하고 전문적으로 디자이너에게 전달하는 핵심 단계입니다.

| 핵심 기능 | 기술적 구현 목표 | 사용자 경험 (UX) |
| :--- | :--- | :--- |
| **요청 사항 입력** | 사용자는 1단계에서 선택한 **합성 이미지**와 **간단한 코멘트**("앞머리를 더 짧게요", "색깔은 톤 다운")만 입력합니다. | 복잡한 미용 용어 없이 **"대충의 느낌"**만 전달하면 됩니다. |
| **전문 용어 변환** | **LLM(Large Language Model)**이 입력된 요청사항과 1단계 분석 결과를 종합하여 **미용 전문가용 요청서**를 생성합니다. (NLP 기술) | AI가 작성한 최종 요청서를 사용자는 **'확인' 버튼**만 누르면 됩니다. |
| **주의 사항 자동 삽입** | 1단계 모발 분석 결과를 바탕으로 **"디자이너가 주의해야 할 사항"**이 요청서 하단에 자동으로 추가됩니다. (예: "모발 끝 건조함 심함. 열펌 시 약제 조절 필수.") |

### 4단계: 예약 및 자동 요청서 전송 (자동화 서비스)

예약과 동시에 모든 정보가 미용실에 전달되어, 미용사가 시술 준비를 완벽하게 할 수 있게 합니다.

| 핵심 기능 | 기술적 구현 목표 | 사용자 경험 (UX) |
| :--- | :--- | :--- |
| **예약 진행** | 2단계에서 선택한 미용실에 실시간으로 예약 API를 통해 예약 요청을 보냅니다. | 예약이 확정되면, 사용자에게 **앱 알림 및 문자**로 즉시 안내합니다. |
| **요청서 자동 전송** | 예약 확정 시, **3단계에서 생성된 요청서**가 담당 디자이너의 대시보드 또는 카카오톡/문자로 **자동 전송**됩니다. | 사용자는 예약 완료 후 더 이상 아무것도 할 필요가 없습니다. |
| **디자이너 사전 인지** | 디자이너는 고객이 도착하기 전에 고객의 **얼굴형, 현재/희망 스타일, 모발 상태, AI 요청서**를 모두 파악할 수 있게 되어, 시술 만족도가 향상됩니다. |
