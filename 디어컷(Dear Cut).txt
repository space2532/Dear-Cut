## 💇‍♀️ Dear Cut (디어컷) 서비스 요약: AI 기반 헤어스타일 컨시어지 & 매칭 플랫폼

Dear Cut은 사용자의 스타일링 고민과 디자이너와의 소통 오류를 해결하는 **AI 기반 미용실 예약/매칭 서비스**입니다. Vision AI와 LLM 기술을 활용하여 사용자에게 최적화된 스타일을 제안하고, 비전문적인 요청을 **전문가용 시술 요청서**로 자동 변환하여 디자이너에게 전달합니다.

---

## 🚀 핵심 기능 (Key Features)

### 1. AI 스타일링 분석 및 시각화 (Personalization Engine)

사용자의 현재 상태를 초정밀 분석하고, 최적의 스타일을 시각적으로 제안하는 개인화 엔진입니다.

* **Vision AI 분석**: 사진 한 장으로 **얼굴형**, **이목구비 비율**, **모발 손상도(상/중/하)**를 분석합니다.
* **GAN 기반 가상 스타일링**: 분석 데이터와 트렌드를 결합하여 사용자에게 가장 잘 어울리는 스타일 5가지를 추천하고, 사용자 얼굴에 **합성(Virtual Try-On)**하여 보여줍니다. * **현실적 조언**: "곱슬이 심해 관리가 어렵습니다"와 같은 스타일별 **제약 조건**을 미리 안내하여 고객의 기대를 관리합니다.

### 2. 스마트 미용실 매칭 (Booking Engine)

단순 위치 기반이 아닌, **'내가 원하는 스타일'을 가장 잘하는** 디자이너를 찾아주는 예약 엔진입니다.

* **스타일별 전문 매칭**: 레이어드 컷, 옴브레 등 **특정 시술에 대한 포트폴리오 데이터**를 기반으로 미용실을 추천합니다.
* **다중 필터링**: 거리순, 평점순 외에 **'AI 추천 스타일 시술 건수'** 기준으로도 정렬할 수 있습니다.
* **실시간 예약**: 빈 시간표 확인 후 원클릭으로 예약이 가능합니다.

### 3. LLM 기반 요청서 자동 생성 (AI Communicator)

사용자의 모호한 요청(예: "대충 이런 느낌으로...")을 미용 전문가가 이해할 수 있는 구체적인 시술 요청서로 변환합니다.

* **자연어 처리(NLP)**: 사용자가 입력한 "앞머리 짧게, 톤 다운" 같은 간단한 코멘트와 합성 이미지를 분석합니다.
* **전문 용어 변환**: LLM(대규모 언어 모델)이 분석 내용을 해석하여 **구체적인 시술 요청서**를 작성합니다.
* **주의사항 자동 기입**: 1단계에서 분석된 **모발 손상도** 정보가 디자이너 주의사항으로 자동으로 포함됩니다.

### 4. 자동화된 예약 및 전송 (Automated Service)

예약 확정 시 필요한 모든 정보를 디자이너에게 즉시 전달하여 시술 준비를 자동화합니다.

* **자동 전송 시스템**: 생성된 **AI 요청서**가 디자이너의 대시보드나 메신저로 즉시 전송됩니다.
* **사전 인지**: 디자이너는 고객 도착 전 **얼굴형, 희망 스타일, 모발 상태**를 미리 파악하여 시술 만족도를 극대화할 수 있습니다.

## 🛠 시스템 아키텍처 및 기술 스택

### 시스템 아키텍처 (System Flow)

서비스는 사용자의 사진 업로드부터 디자이너에게 정보가 전달되기까지 4단계의 자동화된 흐름으로 진행됩니다.

1.  **AI Vision Analysis**: 사용자가 사진을 업로드하면 Vision AI가 얼굴/모발 데이터를 분석합니다.
2.  **GAN Style Synthesis**: 분석 데이터를 바탕으로 GAN 모델이 가상 스타일링 이미지(Virtual Try-On Img)를 생성하고, 사용자가 스타일을 선택합니다.
3.  **LLM Request Generator**: 사용자가 선택한 스타일 데이터와 간단한 코멘트를 바탕으로 LLM이 **전문적인 시술 요청서(Professional Spec Sheet)**를 생성합니다.
4.  **Reservation & Auto-Send**: 예약과 동시에 생성된 요청서가 디자이너 대시보드(Designer Dashboard)로 자동 전송됩니다. 
### 기술 스택 (Tech Stack) (예시)

| 영역 | 기술 스택 | 주요 용도 |
| :--- | :--- | :--- |
| **Frontend** | React Native (or Flutter), Next.js | 모바일 앱 및 웹 구현 |
| **Backend** | Python **FastAPI**, Django | 서버 로직 및 API 엔드포인트 구현 |
| **AI/ML** | OpenCV, PyTorch, StyleGAN2 / Stable Diffusion, **OpenAI API (GPT-4)** or LangChain | 얼굴/모발 분석, 가상 스타일링, 요청서 생성 |
| **Database** | PostgreSQL, Redis | 데이터 관리 |
| **Infrastructure** | AWS (EC2, S3, Lambda) | 클라우드 인프라 구축 |
